Kashiwazaki SEO Poll マニュアル

WordPress でアンケート（投票）を作成・設置し、集計結果を円グラフで表示できるプラグインです。さらに、集めたデータを CSV・XML・YAML・JSON・SVG の5形式で自動生成し、Google データセット検索に対応した構造化データとして公開できます。このマニュアルは、はじめての方でも迷わないよう、画面付きで一つずつ手順を解説します。

このマニュアルの読み方

上から順に読み進めれば、インストール → アンケート作成 → 表示 → データ公開まで一通り理解できる構成になっています。特定の機能だけ知りたい場合は、左のメニューから該当ページへ直接お進みください。

主要機能

アンケート投票

単一選択・複数選択のアンケートを作成し、ショートコード [tk_poll id="123"] で任意の記事・固定ページに設置。投票結果は Chart.js の円グラフでリアルタイムに表示されます。IP アドレスと Cookie による重複投票防止付き。

5形式のデータ自動生成

投票が入るたびに、集計データを CSV・XML・YAML・JSON・SVG の5つのファイル形式で自動生成。/datasets/ 配下の専用ページから閲覧・ダウンロードできます。

SEO 構造化データ

Schema.org の Dataset 型 JSON-LD、Dublin Core メタタグ、パンくずリスト構造化データ、専用サイトマップを自動出力。Google データセット検索での発見性を高めます。

このプラグインでできること

  • 記事・固定ページにアンケートフォームを設置し、来訪者に投票してもらう
  • 投票結果を円グラフ（割合ラベル・凡例付き）で見せる
  • 同じ人による重複投票を、IP アドレスと Cookie の組み合わせで抑制する
  • 集計データを5つのファイル形式（CSV / XML / YAML / JSON / SVG）で配布する
  • Google データセット検索に必要な構造化データ・専用サイトマップを自動生成する
  • どの記事にアンケートを設置したかを管理画面で自動追跡する
